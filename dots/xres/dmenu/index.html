I use dmenu2, which is a patched version of dmenu. The main reason i use this instead of vanilla dmenu is so that I can set its appearance in Xresources

Github dots/xres/dmenu
dmenu.selforeground: bg dmenu.background: bg dmenu.selbackground: fg dmenu.foreground: fg dmenu.font: fnt3 rofi.font: FixedFixedsys 12 rofi.theme: oneliner
