01 Oct 2017 19:13 | budRich | installation guide

These are the steps I take to setup my work environment on a fresh install of Peppermint OS 8. Most of the steps are valid on other GNU/Linux distributions, but I don’t recommend anyone to blindly follow and install everything.

Before doing anything else i start syncing Dropbox. I start with this step because it can take up to 45 minutes, so while Dropbox is syncing there is time to install and setup other programs.

remove unwanted programs

sudo apt remove transmission

add PPA

This PPA gives us access to qt5ct, that is needed for GTK themes to apply to Qt5 applications.
sudo add-apt-repository ppa:nilarimogard/webupd8

upgrade system

sudo apt update && sudo apt upgrade -y

download some third party applications

Vivaldi, Hugo and Sublime. Install packages using:
sudo gdebi after the update is done.

wget stuff for i3

wget https://github.com/Airblader/xcb-util-xrm/releases/download/v1.2/xcb-util-xrm-1.2.tar.gz  
wget https://github.com/i3/i3/archive/4.14.1.tar.gz

sudo freepass

Create a file in /etc/sudoers.d with the following command: sudo visudo -f /etc/sudoers.d/bud
add this line: (replace USER_NAME)
USER_NAME ALL=(ALL) NOPASSWD: /bin/mv, /sbin/reboot, /sbin/shutdown hit Ctrl+x to save and exit nano. Now password is not needed when performing:
sudo mv/reboot/shutdown

When the update is done, install stuff!

dependencies

sudo apt install libglib2.0-dev libgdk-pixbuf2.0-dev \
libxml2-utils ruby-dev python-pip libxcb1-dev \
libxcb-keysyms1-dev libpango1.0-dev libxcb-util0-dev \
libxcb-icccm4-dev libyajl-dev libstartup-notification0-dev \
libxcb-randr0-dev libev-dev libxcb-cursor-dev \
libxcb-xinerama0-dev libxcb-xkb-dev libxkbcommon-dev \
libxkbcommon-x11-dev xutils-dev autoconf automake libtool \
fonts-inconsolata xfonts-terminus qt4-qtconfig qt5ct curl \
libnotify-bin git x11proto-core-dev libx11-dev libxt-dev \
libxfixes-dev libxi-dev libevent-perl libio-async-perl \
libpoe-perl libtask-weaken-perl

applications

sudo apt install inkscape rxvt-unicode-256color feh lolcat \
toilet mpd mpc ncmpcpp mpv qbittorrent spacefm calibre klavaro \
gcolor2 irssi htop newsbeuter rofi suckless-tools parcellite \
xclip xdotool sxhkd fortune compton hsetroot slop maim \
checkinstall numlockx xcape

sudo gem install sass compass 
sudo pip install pygments

Follow this guide to install i3wm

git stuff

https://github.com/jcs/xbanish.git

Numix Solarized GTK theme

Download and unzip the latest release. enter the folder in a terminal and run: sudo make install

To apply theme to QT applications, add the following line:
QT_QPA_PLATFORMTHEME=qt5ct
To ~/.profile & /etc/environment

Launch qtconfig-qt4 and qt5ct and set both to “GTK+”. Then do the same as sudo.

remove unneeded packages

sudo apt autoremove

disable lightDM

edit /etc/default/grub by removing splash from the GRUB command line. Then run these commands to disable lightDM:

sudo update-grub  
sudo systemctl disable lightdm.service  
sudo systemctl enable multi-user.target --force  
sudo systemctl set-default multi-user.target

To temporarily start lightDM:
sudo systemctl start lightdm.service

To enable lightDM just enter these commands:

sudo systemctl enable lightdm.service  
sudo systemctl set-default graphical.target

when Dropbox is in sync

change Dropbox-folder to ~/box/Dropbox (settings menu in systray).

Make sure sublime, Vivaldi and spacefm isn’t running. Remove their config folders if they exist in ~/.config.

run this command: xdg-user-dirs-update
to create ~/.config/user-dirs.dirs and make it sober:

XDG_DESKTOP_DIR="$HOME/tmp"
XDG_DOWNLOAD_DIR="$HOME/dwn"
XDG_TEMPLATES_DIR="$HOME/new"
XDG_PUBLICSHARE_DIR="$HOME/pub"
XDG_DOCUMENTS_DIR="$HOME/dox"
XDG_MUSIC_DIR="$HOME/snd"
XDG_PICTURES_DIR="$HOME/pix"
XDG_VIDEOS_DIR="$HOME/vid"

I have written an article on how to achieve the perfect folder structure.

Follow instructions in ~/etc/seba/README.md to make a keyboard with swedish layout acceptable.

~/bin/dotcrawler install scripts and dots
~/src/igi create an issue

cross your fingers and reboot.
