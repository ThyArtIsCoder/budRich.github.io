25 Sep 2017 17:15 | budRich | fonts urxvt Fixedsys guide

It is easy to get used to bad font rendering. But there are some simple steps one can take to get great looking fonts on GNU/Linux.

Below is a summary of an article on binarytides, with some spice.

download the following fonts from
http://www.google.com/fonts/

  • Droid Sans
  • Droid Sans Mono
  • Droid Serif
  • Noto Sans
  • Noto Serif
  • Open Sans

Unzip ‘em and place them in ~/.fonts. Download and place this: .font.config (it’s a different file from that linked directly in the binary tides article) in your
home folder

I have been using Terminus as my default terminal font for a long time. It looks really stylish, but it is a bit thin and recently i switched to the classic Fixedsys . However it needs some tinkering to render properly in the terminal, just follow this guide:

https://askubuntu.com/questions/210283/how-to-use-fixedsys-in-the-gnome-terminal-or-wherever-monospaced-fonts-are-requ

terminal-shot1.png

I use urxvt and I had to add this line:
printf '\e]710;%s\007' "xft:Fixedsys Excelsior 3.01\-L2 Mono:style=Regular"
to my .bashrc to trick it into using the font.
