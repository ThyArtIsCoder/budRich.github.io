03 Jan 2018 10:22 | budRich | i3

I have gotten some requests on an isolated version of i3fyra’s tabswitching function. And this morning I sat down to try a new way to do it, and it worked .. pretty, pretty, pretty good :)

Vanilla i3 doesn’t have a separate function for switching tabs, instead we use the normal focus [left|right]. This is not so well suited for tabs since if one focus right when the last tab is active, the focus will pop out to the container to the right if there is one.

In i3fyra i managed to get the desired tabswitching function (warp the focus within the tabbed container), it is easy since the four containers are known and marked. And I always thought that the solution to make an isolated version of the function that would work without i3fyra would require either marks or some complicated JSON parsing.

But with the parsing technique i use in i3flip it is done with very little code, no dependencies and no marks. The function is more then twice as fast as the one I used in i3fyra (23 vs 52 ms.), so I will remove that part from the i3fyra source.

i3flip works on all types of layouts not only tabbed and should work well on both vanilla i3, i3-gaps and off course i3fyra . It is a single bash script with no dependencies, I hope you will enjoy it!
