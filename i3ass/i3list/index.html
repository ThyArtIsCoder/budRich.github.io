18/01/03 - window geometry

Added some new output values (45-55) to return the window geometry. Also AWM/TWM (mark) is deprecated, since spaces in marks caused trouble.

i3list - List attributes of i3wm

SYNOPSIS

i3list [OPTION [CRITERIA]]

DESCRIPTION

This script is vital for i3fyra to work. But it can be used on it’s own or with other scripts. i3list is also used by i3run. i3list parses the output of the command:
i3-msg -t get_tree

and returns a long string with 43 values separated by spaces. This strange output format is made to make it possible to pipe the output to other scripts. A lot of the information is specific to i3fyra, but if i3list doesn’t find a value an X will be placed in it’s place in the output.

The idea, is to use this script once, and get all the info about i3’s current state in one big chunk. So one doesn’t have to make more requests for tree output. It have speeded up my own scripts a lot. And it also makes many listener scripts unnecessary. The parsing is done with awk, and it is very fast, about 25ms on my average computer (i5, 4GB). Same operation with pure bash would take at least a whole second (x40).

I have made another script, i3get, that works in a similar way but where you choose which information you want in the output. i3get is slightly faster (5ms), but i3list outputs more information and is meant to replace multiple i3get requests.

OPTIONS

-v Show version info and exit

-h Show this help and exit

-c CLASS Search for windows with the given CLASS

-i INSTANCE Search for windows with the given INSTANCE

-t TITLE Search for windows with the given TITLE

-n CON_ID Search for windows with the given CON_ID

-d WINDOW_ID Search for windows with the given WINDOW ID

-m CON_MARK Search for windows with the given CON_MARK

If no option is passed, active window will be target.

The best way to handle the output is to put it in an array.

Example:

$ list_array=($(i3list))
$ echo ${list_array[42]}

#!/bin/bash

NAME="i3list"
VERSION="0.126"
AUTHOR="budRich"
CONTACT='robstenklippa@gmail.com'
CREATED="2017-10-06"
UPDATED="2018-04-02"




containerlist(){
 
  i3-msg -t get_tree \
  | awk -v RS=',' -F':' \
    -v chkl="$(echo i34{A..D})" -v crit="${crit}" -v srch="${srch}" \
    '
      BEGIN{
        hit="0";trg="0";act="0";marko="X";wsdchk="0"
        VIS="X";HID="X";EXS="X"
        split(chkl,alst," ",seps)
        
        SDW="X";SDH="X";CWS="X"
        FAC="X";FBD="X";WSF="X"

        SAB="X";SAC="X";SBD="X"
        CAB="X";CAC="X";CBD="X"

        TWP="X";TWL="X";TWS="X";TWC="X"
        TWW="X";TWI="X";TWM="X"

        AWP="X";AWL="X";AWS="X";AWC="X"
        AWW="X";AWI="X";AWM="X"

        LOA="X";FCA="X";LCA="X";ACA="X"
        LOB="X";FCB="X";LCB="X";ACB="X"
        LOC="X";FCC="X";LCC="X";ACC="X"
        LOD="X";FCD="X";LCD="X";ACD="X"

        AWX="X";AWY="X";AWB="X";AWWIDTH="X";AWHEIGHT="X";
        TWX="X";TWY="X";TWB="X";TWWIDTH="X";TWHEIGHT="X";
        curx="X";cury="X";curb="X";curw="X";curh="X"      
        AID="X";TID="X";curid="X"
      }

      hit!=0 && $0~"{" {hit++}
      hit!=0 && $0~"}" {hit--}
      hit!=0 && $1=="\"focus\"" {
        gsub("[[]|[]]","",$2)
        conta[marko]["focused"]=$2
      }

      $1=="\"rect\"" {curx=$3;rectw=1}
      rectw==1 && $1=="\"y\""{cury=$2}
      rectw==1 && $1=="\"width\""{curw=$2}
      rectw==1 && $1=="\"height\""{sub("}","",$2);curh=$2;rectw=2}

      $1=="\"deco_rect\"" {rectb=1}
      rectb==1 && $1=="\"height\""{sub("}","",$2);curb=$2;rectb=2}

      $1=="\"window\"" {curid=$2}

      $1=="\"type\"" && $2=="\"workspace\"" {wsdchk="1"}
      wsdchk=="1" && $1=="\"width\""  {dim["w"]=$2}
      wsdchk=="1" && $1=="\"height\"" {gsub("}","",$2);dim["h"]=$2;wsdchk="2"}
      $1=="\"num\"" {curws=$2;if(WSF=="chk"){WSF=$2}}
      wsdchk=="2" && $1=="\"num\"" {
        dim[$2]["w"]=dim["w"]
        dim[$2]["h"]=dim["h"]
        wsdchk="0"
      }

      $1~"{\"id\"" || $2~"\"id\"" {
        wid=$NF
        if(trg=="0" || act=="0"){cid=$NF}
        if(hit!=0){
          conta[marko]["childs"]=conta[marko]["childs"] wid " "
          conta[marko]["id"]=cid
        }
      }
      
      trg=="0" && $1==crit && $2 ~ srch  {TWI=cid;trg="1";TWW=curws}
      trg=="0" && $2 ~ crit && $3 ~ srch {trg="1";TWI=cid;TWW=curws}
      act=="0" && $1=="\"focused\"" && $2=="true" {act="1";AWI=cid}
      numchk=="1" && $1=="\"width\""  {dim[curws]["w"]=$2}
      numchk=="1" && $1=="\"height\"" {gsub("}","",$2);dim[curws]["h"]=$2;numchk=0}
      abchk=="1" && $1=="\"width\""  {CAB=$2;abchk=0}
      acchk=="1" && $1=="\"height\"" {gsub("}","",$2);CAC=$2;acchk=0}
      bdchk=="1" && $1=="\"height\"" {gsub("}","",$2);CBD=$2;bdchk=0}

      $1=="\"floating\"" {
        if(act!="2") {

          if(cid == AWI){
            if($2~"on"){AWP="g"}
            if($2!~"on"){AWP="i"}
            AWW=curws
            AWC="X"
            AWL="u"
            AWS="a"
            if(cid == conta[marko]["id"]){
              AWC=marko
              AWL=conta[marko]["layout"]
            }
            # if(cmrk!=""){AWM=cmrk}
            act="2"
            AWX=curx
            AWY=(cury-curb)
            AWB=curb
            AWWIDTH=curw
            AWHEIGHT=(curh+curb)
            AID=curid
          }
        }

        if(trg!="2") {
          if(cid == TWI){
            if($2~"on"){TWP="g"}
            if($2!~"on"){TWP="i"}
            TWW=curws
            TWC="X"
            TWL="u"
            TWS="n"
            if(TWI == AWI){TWS="a"}
            if(cid == conta[marko]["id"]){
              TWC=marko
              TWL=conta[marko]["layout"]
            }
            # if(cmrk!=""){TWM=cmrk}
            trg="2"
            TWX=curx
            TWY=(cury-curb)
            TWB=curb
            TWWIDTH=curw
            TWHEIGHT=(curh+curb)
            TID=curid
          }
       }
       cmrk=""
     }

      $1=="\"marks\"" {
        gsub("[[]|[]]|\"","",$2);
        for (m in alst) {
          if(alst[m] == $2){
            hit=1
            gsub("i34","",alst[m])
            marko=alst[m]
            conta[marko]["ws"]=curws
            conta[marko]["layout"]="-"
          }
        }

        cmrk=$2

        if($2~"i34XAB"){
          WSF=curws
          if(curws=="-1"){WSF="chk"}
        }

        if($2~"i34SAB"){
          split($2,asplt,"_")
          SAB=asplt[2]
        }
        if($2~"i34SAC"){
          split($2,asplt,"_")
          SAC=asplt[2]
        }
        if($2~"i34SBD"){
          split($2,asplt,"_")
          SBD=asplt[2]
        }
        if($2~"i34FAC"){
          split($2,asplt,"_")
          FAC=asplt[2]
        }
        if($2~"i34FBD"){
          split($2,asplt,"_")
          FBD=asplt[2]
        }
        if($2=="i34XAC"){abchk=1}
        if($2=="i34A"){acchk=1}
        if($2=="i34B"){bdchk=1}
      }

      hit!=0 && $1=="\"layout\"" && conta[marko]["layout"]=="-" {
        gsub("\"","",$2)
        if($2=="tabbed")
          conta[marko]["layout"]="t"
        if($2=="splith")
          conta[marko]["layout"]="h"
        if($2=="splitv")
          conta[marko]["layout"]="v"
        if($2=="stacked")
          conta[marko]["layout"]="s"
      }

      END{
        FS=" "
        if(AWC!="X"){AWP="m"}
        if(TWC!="X"){TWP="m"}
        for (c in conta) {
          $0=conta[c]["childs"]; tmplngth=NF
          split(conta[c]["childs"],achld," ")
          i=0
          for(d in achld){
            i++
            
            if(c~/^A$|^B$|^C$|^D$/){
              if(i == tmplngth){
                if(AWC==c && AWI==achld[d]){AWP="e"}
                if(TWC==c && TWI==achld[d]){TWP="e"}
                conta[c]["LC"]=achld[d]
              }
              if(i==1){
                if(AWC==c && AWI==achld[d]){AWP="b"}
                if(TWC==c && TWI==achld[d]){TWP="b"}
                conta[c]["FC"]=achld[d]
              }
              if(tmplngth==1){
                if(AWC==c){AWP="o"}
                if(TWC==c){TWP="o"}
              }
            }
          }
        }   

        if(TWS!="a"){
          TWS="n"
          if(conta[TWC]["focused"]==TWI){TWS="f"}
        }

        if(TWI==""){
          TWC="X";TWW="X";TWP="X";TWI="X"}

        if(AWI==""){
          AWC="X";AWW="X";AWP="X";AWI="X"}

        for (c in conta) {
          if(c~/^A$|^B$|^C$|^D$/){

            if(conta[c]["ws"]==WSF){gsub("X","",VIS);VIS=VIS c}
            if(conta[c]["ws"]=="-1"){gsub("X","",HID);HID=HID c}

            if(c=="A"){
              LOA=conta[c]["layout"]
              FCA=conta[c]["FC"]
              LCA=conta[c]["LC"]
              ACA=conta[c]["focused"]
            }
            if(c=="B"){
              LOB=conta[c]["layout"]
              FCB=conta[c]["FC"]
              LCB=conta[c]["LC"]
              ACB=conta[c]["focused"]
            }
            if(c=="C"){
              LOC=conta[c]["layout"]
              FCC=conta[c]["FC"]
              LCC=conta[c]["LC"]
              ACC=conta[c]["focused"]
            }
            if(c=="D"){
              LOD=conta[c]["layout"]
              FCD=conta[c]["FC"]
              LCD=conta[c]["LC"]
              ACD=conta[c]["focused"]
            }
          }
        }
        EXS=VIS HID
        if(EXS=="XX"){EXS="X"}
        else{gsub("X","",EXS)}
        if(EXS==""){EXS="X"}
        AWM="X"; TWM="X"
        
        print AWW " " VIS " " HID " " EXS " " \
          AWP " " AWL " " AWS " " AWC " " AWW " " AWI " " AWM " " \
          TWP " " TWL " " TWS " " TWC " " TWW " " TWI " " TWM " " \
          LOA " " FCA " " LCA " " ACA " " \
          LOB " " FCB " " LCB " " ACB " " \
          LOC " " FCC " " LCC " " ACC " " \
          LOD " " FCD " " LCD " " ACD " " \
          SAB " " SAC " " SBD " " \
          CAB " " CAC " " CBD " " \
          FAC " " FBD " " \
          dim[AWW]["w"] " " dim[AWW]["h"] " " \
          WSF " " \
          AWX " " AWY " " AWWIDTH " " AWHEIGHT " " \
          TWX " " TWY " " TWWIDTH " " TWHEIGHT " " \
          AID " " TID " " \
          AWB " " TWB
      }
    '
}

main(){

  while getopts :c:i:t:n:d:r:m:avh option; do
    case "${option}" in
      i) crit="\"instance\"" srch="${OPTARG}";;
      c) crit="\"class\"" srch="${OPTARG}";;
      t) crit="\"title\"" srch="${OPTARG}";;
      n) crit="\"id\"" srch="${OPTARG}";;
      d) crit="\"window\"" srch="${OPTARG}";;
      m) crit="\"marks\"" srch="${OPTARG}";;
      a) crit="\"focused\"" && srch="true";;
      v) printf '%s\n' \
           "$NAME - version: $VERSION" \
           "updated: $UPDATED by $AUTHOR"
         exit ;;
      h|*) printinfo && exit ;;
    esac
  done

  [ -z "$crit" ] && crit="\"focused\"" && srch="true"

  containerlist
}

printinfo(){
  case "$1" in
    m ) printf '%s' "${about}" ;;
    
    f ) 
      printf '%s' "${bouthead}"
      printf '%s' "${about}"
      printf '%s' "${boutfoot}"
    ;;

    ''|* ) 
      printf '%s' "${about}" | awk '
         BEGIN{ind=0}
         $0~/^```/{
           if(ind!="1"){ind="1"}
           else{ind="0"}
           print ""
         }
         $0!~/^```/{
           gsub("[`*]","",$0)
           if(ind=="1"){$0="   " $0}
           print $0
         }
       '
    ;;
  esac
}

about='
`i3list` - List attributes of i3wm

SYNOPSIS
--------

`i3list` [`OPTION` [*CRITERIA*]]

DESCRIPTION
-----------

This script is vital for i3fyra to work. But it can be used on it'"'"'s own
or with other scripts. `i3list` is also used by i3run. `i3list` parses the
output of the command:  
`i3-msg -t get_tree`  

and returns a long string with 43 values separated by spaces. This strange 
output format is made to make it possible to pipe the output to other scripts. 
A lot of the information is specific to i3fyra, but if `i3list` doesn'"'"'t find a 
value an X will be placed in it'"'"'s place in the output.  

The idea, is to use this script once, and get all the info about i3'"'"'s current
state in one big chunk. So one doesn'"'"'t have to make more requests for 
tree output. It have speeded up my own scripts a lot. And it also makes many
listener scripts unnecessary. The parsing is done with awk, and it is very
fast, about 25ms on my average computer (i5, 4GB). Same operation with pure
bash would take at least a whole second (x40).  

I have made another script, `i3get`, that works in a similar way but where you
choose which information you want in the output. `i3get` is slightly faster (5ms),
but `i3list` outputs more information and is meant to replace multiple `i3get` requests.

OPTIONS
-------

`-v` 
  Show version info and exit

`-h` 
  Show this help and exit

`-c` *CLASS*
  Search for windows with the given *CLASS*

`-i` *INSTANCE*
  Search for windows with the given *INSTANCE*

`-t` *TITLE*
  Search for windows with the given *TITLE*

`-n` *CON_ID*
  Search for windows with the given *CON_ID*

`-d` *WINDOW_ID*
  Search for windows with the given *WINDOW ID*

`-m` *CON_MARK*
  Search for windows with the given *CON_MARK*

If no option is passed, active window will be target.
