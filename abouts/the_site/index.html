This site is made using the static site generator Hugo and the text editor Sublime Text, using GNU/Linux as an IDE.

I wanted a static site because cutting edge web development techniques and bloated IDE’s is not an environment I like. I have promised myself that the rendered site should only be HTML/CSS (except for google analytics and disqus). There is a lot of crazy stuff and bash scripting involved in creating the site, but that doesn’t affect the finished site or your RAM and bandwidth usage.

Crazy stuff, bash scripting and using GNU/Linux as an IDE is what I have intended to be the main content of this site. There might be reviews (software, books, podcasts, blogs, music), food recipes, DIY stuff and prepping guides. We’ll see. I can however promise that the only political statement made, will be the site itself.

I will also publish a podcast and videos on the same subjects here soon.
