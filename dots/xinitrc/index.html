This is part of my anti-DM suite. The stuff I want to happen just before i3 loads up, goes here.

Starting urxvt in daemon mode is really nice and makes it consume less resources and start much faster.

I use hsetroot to set the desktop background to a minimal tiled pattern and a fetch color from Xresources.

The Seba keyboard layout keeps me sane!

I use sxhkd to remap the back button of the mouse to backspace.

I used to use unclutter to hide the mouse pointer, but I found it broken. xbanish is a much better alternative, but I had to download and compile it from source, since it’s not in the ubuntu repositories.

compton compositor.

#!/bin/sh
# .xinitrc - curated by budRich

xrdb -merge $HOME/.Xresources
xrandr --output eDP-1 --auto --left-of HDMI-1 --primary &
setxkbmap -layout seba -option ctrl:nocaps -option lv3:ralt_switch
hsetroot -tile /home/bud/etc/bitmap-walls/patterns-png/line_diag3_2.png \
  -tint "$(xrdb -query | awk '/\*background/ {print $2}')"
urxvtd -q -o -f
dropbox start &

igi &

killall polybar
killall qbittorrent
killall mpd

compton -b > /dev/null 2>&1
sxhkd > /dev/null 2>&1 &

xbanish > /dev/null 2>&1 &
mpd > /dev/null 2>&1 &

exec i3
