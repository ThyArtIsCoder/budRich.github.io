You can probably install i3 directly from your distributions package manager. But If you want to be sure to get the latest version or a fork you need to download and compile the source code.

i3-gaps

i3-gaps is a fork of i3 developed by Airblader who happens to be one of the main developers on vanilla/normal i3. i3-gaps has some features that is not 100% in sync with the design philosophy of normal i3, that’s why it is a fork. However it is a very totally stable fork and it has all functionality that vanilla i3 has and it always gets updated when the main branch does. So you are safe to use i3-gaps. The big difference between the two is that i3-gaps has, well, gaps. Meaning there is a gap between the windows which produce pretty screenshots that you can spam r/unixporn with 😜

version 4.14

A new i3 version, 4.14, was released just the other day. When writing this, the main ubuntu repositories have not added this yet, so to obtain the latest release i did this:

install dependencies

sudo apt install libxcb1-dev libxcb-keysyms1-dev \
libpango1.0-dev libxcb-util0-dev libxcb-icccm4-dev libyajl-dev \
libstartup-notification0-dev libxcb-randr0-dev libev-dev \
libxcb-cursor-dev libxcb-xinerama0-dev libxcb-xkb-dev \
libxkbcommon-dev libxkbcommon-x11-dev xutils-dev autoconf \
automake libtool checkinstall

cd /tmp
wget https://github.com/Airblader/xcb-util-xrm/releases/download/v1.2/xcb-util-xrm-1.2.tar.gz
tar xzf xcb-util-xrm-1.2.tar.gz
cd xcb-util-xrm
./autogen.sh --prefix=/usr
make
sudo checkinstall

download, build and install i3

cd /tmp
# get the official release tar of 4.14.1
wget https://github.com/i3/i3/archive/4.14.1.tar.gz
# extract it
tar xfz 4.14.1.tar.gz
cd i3-4.14.1
# generate configure
autoreconf -fi
mkdir build && cd build
../configure --prefix=/usr --sysconfdir=/etc
make
sudo checkinstall

now you have a fresh install of vanilla i3 v:4.14.1 If you rather want gaps, follow this guide by benjames, that I based my guide on.

OBS! This will only install i3. You will probably want a compositor and some sort of launcher and a status bar. Most people use compton together with i3 for composition (this will optimize your screen, reduce screen tearing, and let you set shadows and transparency for windows). dmenu and/or rofi are popular choices for launching. There are many different status bars you can use, the most popular are i3-blocks, polybar or the built in bar with i3-status.

check out this great youtube tutorial on how to get started with i3wm
